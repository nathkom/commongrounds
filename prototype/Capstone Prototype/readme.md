Prototype Overview

This repository contains the prototype for our capstone project. Its purpose is to visualize and translate our wireframes into a semi-functional interface to explore layout, interaction patterns, and overall user experience.
This is not the production version of the product.

Media Usage:
Some images included in this prototype are for mockup purposes only. They are not cleared for commercial or production use and may not be used in a live environment without proper licensing or attribution.

Development Notes:
This prototype was developed with the help of Claude Code, with iterative refinements and design adjustments made manually. While it serves as a strong foundation for interaction patterns and UI direction, the final product should be built as a separate project from the ground up.

The final finished deliverable should:

Re-evaluate architectural decisions

Apply production-ready best practices

Replace placeholder assets with properly licensed media

Refine and validate design and code decisions before implementation
